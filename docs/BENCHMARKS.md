# Benchmarking Standards

Q-MATIS utilizes rigorous benchmarks (Milestone A3-A5 results are documented in the main README).

We evaluate:
1. **MAE/RMSE** on standard holdouts.
2. **Prediction Interval Coverage Probability (PICP)**.
3. **Mean Prediction Interval Width (MPIW)**.
4. **Hardware Footprint**: Parameters, memory, throughput.

Scripts located in scripts/benchmarks/.
